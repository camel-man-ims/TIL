## 1. hashMap vs hashTable

| hashMap                    | hashTable                 |
| -------------------------- | ------------------------- |
| non sync                   | sync                      |
| -                          | legacy                    |
| inherits AbstractMap class | inherits Dictionary class |

* synchronized가 걸리면 thread-safe라는 말이고, 많은 쓰레드에 의해 공유될 수 있다는 말이다.
